# Выгрузка списка фальсификатов из Eprica 5 для импорта в F3Tail

eprica5.xml - файл с настройками импорта
Справочники - фальсификаты - загрузка данных

формат  DBASE3
кодировка Windows1251

в файле EpricaExportFalsificat.exe.config
необходимо настроить путь к бд eprica __eprica.dat__
и путь куда программа будет сохранять __falsificat.dbf__
